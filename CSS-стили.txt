Flexbox - используется для реализации резиновой верстки, позволяя элементам занимать 
доступное пространство и масштабироваться при изменении ширины окна браузера.

Media Queries - применяется для адаптации сайта для печати, скрывая все элементы 
кроме основного контента.

Ширина колонок: Левая колонка (content) занимает все доступное пространство, 
кроме правой колонки (sidebar), которая имеет фиксированную ширину.

Min-height - используется для задания минимальной высоты для блоков "контент" и "доп. информация".

Отступы - применяются для создания визуального разделения между блоками и улучшения читаемости.

SEO-оптимизация - в коде используются семантически правильные теги (header, nav, main, footer), 
которые помогают поисковым системам лучше понять структуру сайта.
